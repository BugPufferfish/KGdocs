进入服务器须知 ​

DANGER

玩家名长度为3-16字符,不能中文,只能包含英文数字大小写及下划线

JAVA版中多人游戏中添加服务器输入地址g1-2.xyeidc.cn:20498

注册及登录 每个玩家第一次进入服务器后都需要注册

若出现( 请输入 /login 密码 来登录游戏 )

则说明该游戏名字已被其他玩家注册

此时你需要换一个游戏名字重新进入游戏

若出现( 使用 /reg 密码 重复密码 完成注册,密码由您自行决定 )

则说明该游戏名字你可以注册

此时你只需要输入/reg 123123 123123 即可完成注册

下次登录时你只需要输入/login 123123 即可完成登录

123123 只是举例密码 请自行取更复杂的密码

最后,在进入服务器之前请仔细阅读服务器规则,以避免不必要的纠纷
